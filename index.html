projects

Things I've built.

PhishGuard → github
Full-stack mobile application that detects phishing threats in real-time from text and image inputs. Built in a 6-person team using Flutter and Firebase. I designed the UI for Previous Inquiries and Settings screens, architected the Firestore schema, and built the DatabaseService layer for all CRUD operations.
Flutter Dart Firebase Firestore Security Mobile
Multiplayer Quiz — TCP Network App → github
Real-time multiplayer quiz game built from scratch on raw TCP sockets. Designed a custom text-based protocol for game state sync, question delivery, and scoring. Implemented thread-safe concurrent answer processing with mutex locks to eliminate race conditions.
C++ TCP Sockets Multithreading Mutex Tkinter GUI Networking
Clustering Malware Families — Static Analysis → github
Unsupervised ML pipeline that clusters malware into families using only static metadata — no execution required. Curated 500 samples via MalwareBazaar and VirusTotal APIs, applied PCA + K-Means, achieving Silhouette Score 0.53 and NMI 0.55.
Python K-Means PCA Malware Analysis VirusTotal API Cybersecurity
LLM Bias Analysis — Independent Research → github
Empirical study of representational bias (gender, ethnicity, orientation) in GPT-4o and Claude 3.7 using a two-phase engineered prompting approach. Automated data collection and demographic extraction over 200+ model responses. Revealed persistent bias patterns despite mitigation layers.
Python LLMs GPT-4o Claude 3.7 Data Analysis Research
Head Pose & Gaze Estimation — Vision Pipeline
Guided research reproducing and extending a computer vision pipeline for gaze estimation. Integrated SynergyNet (3D head pose), YOLOv5 (object detection), and MiDaS (monocular depth) to generate gaze vectors from face to observed object.
Python YOLOv5 SynergyNet MiDaS Computer Vision Depth Estimation
